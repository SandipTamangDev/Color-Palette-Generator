============= Version 1 ===============

Color Palette Generator

Overview:
- Generate colors for 5 boxes.
- Supports random colors and harmonious colors using HSL (Hue, Saturation, Lightness).
- Colors can be shown in Hex, RGB, or HSL.

Features:
1. Five Color Boxes
   - Displays 5 distinct colors.

2. Random Colors
   - Generates true random colors (hex).
   - Converted to RGB for display.

3. Harmonious Colors (HSL)
   - Hue: 0-360° (determines base color)
   - Saturation: 0-100% (0=gray, 100=vivid)
   - Lightness: 0-100% (0=black, 50=normal, 100=white)
   - Fix Saturation and Lightness, let user choose Hue.
   - Ensures colors are in harmony.

4. User Options
   - Choose Random or Harmonious colors.
   - Adjust Hue, Saturation, Lightness if desired.
   - View colors in Hex, RGB, HSL.

5. Error Handling
   - Prevent duplicate colors in boxes.
   - If duplicate occurs, regenerate a new color.

Color Generation Logic:
- Random: generate hex → convert to RGB → display.
- Harmonious: generate HSL → convert to RGB → convert to Hex → display.
- Check duplicates across boxes → regenerate if needed.

Workflow:
1. User selects color type (Random or Harmonious).
2. Optional sliders for H, S, L.
3. Generate colors based on selection.
4. Check for duplicates → regenerate if needed.
5. Display colors in Hex, RGB, HSL.

Future Enhancements:
- Save favorite palettes.
- Preset color schemes (analogous, complementary, triadic).
